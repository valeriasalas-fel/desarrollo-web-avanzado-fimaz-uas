# Práctica 1 - POO en PHP

## Objetivo
Aplicar los fundamentos de la programación orientada a objetos en PHP utilizando encapsulamiento.

## Descripción
Se creó una clase llamada Usuario con los atributos privados:
- nombre
- correo

La clase incluye constructor, getters y setters para acceder a los datos.

## Ejecución

1. Colocar el proyecto en htdocs de XAMPP
2. Iniciar Apache
3. Abrir en el navegador:

http://localhost/desarrollo-web-avanzado-fimaz-uas/parcial-1-poo/practica-1/index.php

# Práctica 4 - POO + Herencia + Validaciones + Excepciones

## Objetivo

Desarrollar un mini sistema en PHP utilizando:

- Programación Orientada a Objetos
- Herencia
- Encapsulamiento
- Validación de datos
- Manejo de excepciones
- Salida en HTML

## Requisitos

- PHP 8+
- XAMPP
- Navegador web

## Ruta de ejecución

Abrir en navegador:

http://localhost/desarrollo-web-avanzado-fimaz-uas/parcial-1-poo/practica-4/index.php

## Evidencia esperada

- Tabla HTML con usuarios
- Usuario Administrador
- Usuario Alumno
- Usuario Invitado
- Mensaje de error controlado cuando se detecta un correo inválido
